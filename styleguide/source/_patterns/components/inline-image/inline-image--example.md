---
title: 'Inline Image'
---
#### Description
The inline image component accepts an image and an optional caption. It allows you to insert images into your WYSIWYG content as you need them. There are three layout options, the default option is full-width, which means the image will take up the full width of the space you place it into. There is also a left and right floated variant.

#### Variables
~~~
src: media / required
alt: string / required
caption: string / optional
orienation: select / required / choices: full (default), left, or right
~~~
